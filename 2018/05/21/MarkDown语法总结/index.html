MarkDown语法总结

  • MarkDown
  • hexo
  • 入门级

Markdown是一种轻量级的『标记语言』，用简洁的语法代替排版。其不追求大而全,只追求易读易写,其简洁的语法成为广受码农欢迎的网络语言.
本篇介绍Markdown基本语法，内容很少，学会后可轻松写出高大上的文档，再也不需要各种编辑器去调文章格式。另外，网上有各平台下的Markdown工具可用，也有在线的，我比较习惯用sublime搞定，Markdown本来就是为了追求简洁，弄个工具岂不多此一举。
